JiFFI2
------
File format interchanger for the Atari Jaguar.


Version 1.0.5 - 08-22-2025
- Added CLI_ONLY define to compile only the command line interface
- Removed the need of libcoff.h
- Fix the size calculation in the ELF format reader
- Removed the need of crc32 library
- Fix missing icon in the Windows executable
- Added version information in the Windows executable
- Set Windows's high DPI scalling override to system

Version 1.0.4 - 07-21-2025
- Fix for gcc compilation
- Added a CMakeLists.txt file for CMake support
- Dispatched the jiffi2.txt file to README.md, Wiki pages and changelog.txt

Version 1.0.3 - 08-20-2024
- Fix potential crash with a VMA & LMA different address in the binary ELF
- Fix COFF export format

Version 1.0.2 - 06-23-2024

Version 1.0.1 - 08-23-2022
